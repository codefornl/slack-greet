In dit kanaal vind je een samenvatting van de informatie die je nodig hebt om een vliegende start te maken :airplane_departure:

Code for NL is de Nederlandse community van developers en designers die samenwerken aan een open, eerlijke, en inclusieve digitale overheid en samenleving. Ons doel is een sterke Nederlandse _public tech_ beweging die vanuit publieke waarden en belangen werkt aan digitale transformatie. Code for NL is onderdeel van het internationale Code for All netwerk. Voor meer informatie, zie www.codefor.nl

*Wat moet je weten voor je begint?*

* Code for NL is een community waarin 'doen' en 'open' centraal staan (https://github.com/codefornl/community-principes)
* Op deze Slack omgeving geldt de Code for NL gedragscode (https://github.com/codefornl/gedragscode)
* Voor vragen en opmerkingen is het bestuur te vinden op <#CDBR8Q56W|bestuur-publiek>
* Wij onderschrijven de Standard for Public Code (https://standard.publiccode.net/)
* Niet spammen
